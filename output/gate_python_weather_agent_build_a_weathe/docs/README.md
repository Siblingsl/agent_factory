                ## Quick Start

                ### 1. Install
                ```bash
                pip install -r requirements.txt
                ```

                ### 2. Configure Env
                ```bash
                cp .env.example .env
                export WEB_SEARCH_API_KEY=...
export CODE_EXEC_API_KEY=...
                ```

                ### 3. Run
                ```bash
                python agent.py
                ```
